## 🛠 Описание
**Hasher-Core** — это универсальное криптографическое ядро, написанное на Python. Инструмент позволяет быстро генерировать хеши различных стандартов: от классического MD5 до сверхнадежного SHA-512. Главная фишка — модуль динамической "присолки" для защиты паролей от брутфорса.

## 💎 Особенности
*   ✅ **Multi-Hash**: Поддержка MD5, SHA-1, SHA-256, SHA-512.
*   🧂 **Dynamic Salt**: Модуль с 17+ уникальными солями и префиксом `0` для усложнения взлома.
*   🚀 **Zero Dependencies**: Работает на стандартной библиотеке `hashlib`, не требует лишних установок.
*   💻 **UX**: Цветовая индикация в терминале и удобное меню управления.
*   🔒 **Secure Exit**: Корректный выход через `sys.exit` или `Ctrl+C`.
